Vorgänge:
1.  Neuanlegen eines Events
	Aufruf der new Action beim Formular öffnen creat-Action beim Speichern des Formulars

	Routing:
		new_event_path - Methode
		event_path, method: post

2. Editieren eines Items
	Aufruf der edit - Action
		edit_event_path(id)
	update-Actiom: Speichern der Änderungen

3. View-Bereich:
	Im Ordner Views, Unterordner events gibt es eine Datei new.html.erb und eine Datei edit.html.erb, die ein Partial aufrufen, in dem das eigentliche Formular definiert ist.
  
4. Form-Helper
	form_for - Anlegen eines formulars
	<%= form_for @ event do |f| %> ..... <% end %>
	text_field - Einfache Textfeld (1 Zeile)
	<%= f.text_field :name %>
	...

	5. Conroller:
		def create
			@event = Event.new(event_params)

		Diese Methode enthält alle erlaubten Parameter. Dies ist ein Rials-Sicherheitsfeature
